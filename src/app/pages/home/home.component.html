Welcome to ChemApp!

       

Secure Access to Chemical Compound Data

     
           
       

          ChemApp is your portal for browsing, searching, and managing **essential chemical compound data**. This application is built using **Angular Standalone Components** and features a robust, role-based security system powered by **JWT Authentication**.        

               

Key Features

       
             
  • Browse a **paginated list** of chemical compounds in a clean card view.
    •          
  • View **full details** for any compound by clicking its card.
    •          
  • **Secured Access:** Data endpoints are protected, accessible only after login.
    •          
  • **Admin Privileges:** Users with the **Admin role** gain access to management tools, including **editing** and **deleting** compounds, and **creating** new user accounts.
